顺序表的实现

0、C++ 源文件

顺序表.cpp

1、顺序表功能描述

顺序表相当于你在维护一个动态的一维数组。

顺序表应实现以下功能：

  1. 初始化顺序表：建立一个空的顺序表。
  2. 插入元素：在顺序表的某个位置插入新元素。
  3. 删除元素：删除线性表中某个位置的元素。
  4. 查找元素：在顺序表中查找给定元素。
  5. 修改元素：通过在顺序表中的位置获取元素，并修改元素的值。
  6. 遍历顺序表：输出顺序表的所有元素。
  7. 清空顺序表：实现顺序表的逻辑清空
  8. 统计元素个数

2、菜单功能

2.1 功能描述

  • 提供一个可供用户选择的界面

2.2 分析

  1. 在菜单函数中输出功能页面。
  2. 在主函数中利用 switch 语句将功能分段。

  1. 在 switch 语句中调用 menu() 函数，便于用户进行选择

3.1 功能描述

3.2 分析

  • 使用语句 exit(0) 进行退出。

3.3 步骤

  1. 在 switch 语句中调用 exitSqList() 函数

4、初始化顺序表

4.1 设计顺序表的数据结构

  1. 存储空间的起始位置：数组data
  2. 线性表的最大存储容量：数组长度MAXSIZE
  3. 线性表的当前长度：length

4.2 实现初始化函数

初始化顺序表是指给顺序表的数据（data和length）赋予实际意义。

  • 给 data 分配 MAXSIZE 的空间
  • 给 length 赋值为0
  1. 封装一个 InitSqList(&L) 函数
  1. 在 main 中测试 InitSqList(&L)函数

5、插入元素

5.1 功能描述

  • 在线性表中的某个位置插入新元素。

5.2 分析

  1. 扩容判断

    • 如果插入之前 length 小于 capacity 则不需要扩容。

    • 如果插入之前 length 等于 capacity 则需要扩容。

  2. 扩容

    • 新建指针，指向更大的内存，例如：int *p = new int[MAXSIZE + 10];

    • 更新容量。

    • 将原数组赋值给新的数组。

    • 将 data 指针指向新数组。

    • 释放原指针内存。

  3. 插入位置

    • 如果插入位置 i 比第1位小，比最后1位加1还大时，返回false。

    • 如果插入位置 i 不在表尾，则需要将 i 及其之后的元素向后移动一个位置，

    • 将元素插入到 i 的位置。

    • 插入成功将 length + 1。

注意：i 不是下标。

5.3 步骤

  1. 封装 bool SqListInsert(SqList &L, int i, int e) 函数
  1. 在 switch 语句中测试 SqListInsert(SqList &L, int i, int e) 函数

6、遍历顺序表

6.1 功能描述

  • 输出顺序表的所有元素

6.2 分析

  1. 如果顺序表中没有元素则输出没有元素【此处封装一个顺序表是否为空的函数】
  2. 如果顺序表中有元素，则输出顺序表中的全部元素

6.3 步骤

  1. 封装 bool SqListEmpty(SqList L) 函数
  1. 封装 void showSqList(SqList L) 函数
  1. 在 switch 语句中进行测试

7、删除元素

7.1 功能描述

  • 删除线性表中某个位置的元素。

7.2 分析

  1. 判断删除位置 i 是否合法。
  2. 用 e 接受被删除的元素。
  3. 将 i 之后的元素向前移动一个位置。
  4. 将顺序表的当前长度减1。

7.3 步骤

  1. 封装 bool SqListDelete(SqList &L, int i, int &e) 函数
  1. 在 switch 语句中测试 SqListDelete(SqList &L, int i, int &e) 函数

8、查找元素

8.1 功能描述

  • 在顺序表中查找给定元素，如果查找成功，返回该元素返回给 e；否则，返回0表示失败

8.2 分析

  1. 查找所给值是否在顺序表中，找到则返回其下标。
  2. 如果在顺序表中，则查找其下标，并将其返回给e。

8.3 步骤

  1. 封装 int LocateElem(SqList L, int e) 用于查找所给值是否在顺序表中，是则返回其下标
  1. 封装 void FindElem(SqList L, int e) 用于通过下标返回顺序表中的元素
  1. 在 switch 语句中测试 FindElem(SqList L, int e) 函数

九、修改元素

9.1 功能描述

  • 通过在顺序表中的位置获取元素，并修改元素的值。

9.2 分析

  1. 通过位置 i 获取元素
  2. 判断 i 是否合法。
  3. 取得该元素，并修改其值。

9.3 步骤

  1. 封装 bool GetElem(SqList L, int i, int &e) 函数，用于获取位置为 i 的元素。
  1. 封装 void UpdateElem(SqList L, int i, int e) 函数，用于修改元素的值
  1. 在 switch 语句中进行测试
